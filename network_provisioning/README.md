# Network Provisioning component

[![Component Registry](https://components.espressif.com/components/espressif/network_provisioning/badge.svg)](https://components.espressif.com/components/espressif/network_provisioning)

This component extends [wifi_rovisioning component](https://github.com/espressif/esp-idf/tree/master/components/wifi_provisioning) to encompass broader network provisioning, including Thread provisioning. Both the Wi-fi devices and Thread devices can be provisioned into network with the esp_prov.py script.
